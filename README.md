# e-Appointment Management System
This project is a professional appointment scheduling application designed to streamline healthcare bookings. It serves as a practical implementation of full-stack development concepts, focusing on clean architecture and seamless user experience.
# Key Learning Outcomes & Implementation
During the development of this project, I have implemented and mastered the following technical concepts:

Angular Fundamentals: Developed a structured frontend using Module-based architecture and handled user inputs via Two-way Data Binding ([(ngModel)]).

Real-time Filtering: Implemented a dynamic search system to filter doctor lists instantly without page reloads.

Responsive UI/UX: Built a mobile-first, professional interface using Bootstrap 5 and FontAwesome icons.

Secure Backend Integration: Connected the frontend to a .NET Web API backend, managing data flow with HttpClient and RxJS.

Identity & Security: Gained experience in handling secure user sessions and authentication flows using JWT.

Database Management: Managed complex data relations and CRUD operations via Entity Framework Core and SQL Server.
# Tech Stack
Frontend: Angular (Modules, Components, Forms)

Backend: ASP.NET Core Web API

Database: MS SQL Server

Styling: Bootstrap 5, CSS3
